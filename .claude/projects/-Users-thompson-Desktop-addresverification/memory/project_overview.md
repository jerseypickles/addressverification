Shopify embedded app for address verification at checkout.

- **API**: Google Address Validation API
- **Type**: Embedded Shopify app
- **Stack**: Node.js / Remix (Shopify standard)
- **Features**: Validate addresses AND suggest corrections
- **Market**: USA only

**Why:** Reduce failed deliveries, returns, and extra shipping costs by validating shipping addresses before order processing.

**How to apply:** All architecture decisions should target US addresses only, use Google Address Validation API, and follow Shopify's embedded app patterns with Remix.
